# DriveRent — Система бронювання автомобілів

## Опис проекту

DriveRent — це веб-застосунок для автоматизації прокату авто. Система дозволяє клієнтам обирати машини на потрібні дати, а адміністраторам — керувати парком авто та замовленнями.

## Технологічний стек

- **Backend:** Node.js, TypeScript, Express.
- **Frontend:** React.
- **Database:** PostgreSQL.
- **Testing:** Jest.
- **DevOps:** Docker, GitHub Actions.

## Основний функціонал

- **Каталог авто**: Перегляд списку машин із технічними характеристиками та цінами.
- **Бронювання**: Можливість вибрати період оренди та забронювати конкретне авто.
- **Керування статусами**: Автоматична зміна стану автомобіля (Доступний / Орендований / На обслуговуванні).
- **Калькулятор вартості**: Динамічний розрахунок ціни залежно від тривалості оренди.
